This is a simple informational website about the history of architecture from the Neolithic period to modern times. It is built using JavaScript with Three.js and fflate libraries. The site features 5 architectural 3D models. Navigation is performed using mouse controls. The website includes a color theme switching functionality.

To launch this website, you will need Express. You can install it using npm.

To start the server, open Command Prompt, navigate to the website directory, and run: node server.js. After that, copy the generated link and navigate to it. The address should look something like this: link/pages/index.html
